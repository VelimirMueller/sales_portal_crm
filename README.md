# 🚀 Getting Started – Sales Portal CRM

A minimal CRM frontend built with **React**, **Vite**, and **Tailwind CSS**.

---

## ⚙️ Prerequisites

- [Node.js](https://nodejs.org/) (v18+ recommended)
- [pnpm](https://pnpm.io/) or `npm` / `yarn`

---

## 📦 Install

```bash
pnpm install
# or
npm install
```
